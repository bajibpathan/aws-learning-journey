# 📖 Overview

Amazon Simple Storage Service (Amazon S3) is AWS's highly durable and scalable object storage service designed to store virtually unlimited amounts of unstructured data.

Amazon S3 is widely used for:

- Static website hosting
- Backups and disaster recovery
- Data lakes
- Media storage
- Log storage
- Cloud-native applications
