# 🧪 Amazon S3 Hands-on Labs

This directory contains the hands-on labs completed while learning Amazon S3.

Each lab demonstrates real AWS implementation, validation, troubleshooting, and production best practices.

---

# 📋 Lab Structure

Every lab follows a consistent structure:

- 🎯 Objective
- 📋 Prerequisites
- 🏗 Architecture
- ⚙️ Implementation Steps
- ✅ Validation
- 💻 Commands Used (if applicable)
- ⚠️ Issues Encountered
- 💡 Lessons Learned
- 🔒 Production Best Practices
- 🎯 Result

# 🎯 Learning Outcomes

After completing these labs, you will be able to:

- Create and manage Amazon S3 buckets
- Secure bucket access using IAM and Bucket Policies
- Configure Versioning
- Restore deleted objects using Delete Markers
- Configure Same-Region and Cross-Region Replication
- Validate replication
- Apply production best practices
